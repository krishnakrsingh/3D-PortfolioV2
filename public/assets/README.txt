# Assets Directory

This folder contains the media assets used in the application.

## What goes here?
- **Images**: PNG/JPG files for your projects and profile.
- **Audio**: `music.mp3` used in the Hero section vinyl player.

## Current Files

- `*.png`: Placeholder images for the project grid.
